# Quack engine

## Dependencies

- Cmake

## Clone with
```shell
git clone --recurse-submodules -j8 git@gitlab.com:quack-engine/engine.git  
```
## Update submodules
```shell
git submodule update --init --recursive
```

### Use CLion

## Build process with cmake (manual)
### Make build directory
```shell
mkdir cmake-build-debug
```
### Configure project

```shell
cd cmake-build-debug
cmake ..
```

### Build
```shell
cmake --build .
```
